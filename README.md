Ruby+Phidgets
=============

* Ruby 1.8.7+
* Mac OSX
* [Phidgets](http://www.phidgets.com)

Install Dependencies
--------------------

Install [Phidget21 Library](http://www.phidgets.com/drivers.php)

Install rubygem

    % gem install ffi phidgets-ffi


Run
---

    % ruby ffi-ifkit.rb
    % ruby ffi-rfid.rb
    % ruby ffi-servo.rb
